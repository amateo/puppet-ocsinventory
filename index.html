Tested on...

  • CentOS 7

Example usage

Install OCSInventory

node /box/ {
  include ocsinventory
}

Configure OCSInventory

node /box/ {
  class { 'ocsinventory':
    agent_ocs_mode   => 'cron',
    agent_ocs_pause  => '100',
    agent_ocs_server => 'localhost',
    agent_ocs_tag    => undef,
  }
}
